## Deskripsi Peran
Sebagai Front-End Developer, saya bertanggung jawab atas 
tampilan antarmuka pengguna (UI) pada proyek ini menggunakan 
Laravel Livewire untuk membangun UI yang dinamis dan reaktif 
tanpa perlu menulis JavaScript secara manual.

## Teknologi yang Digunakan
- **Laravel** - Framework PHP sebagai fondasi aplikasi
- **Livewire** - Library untuk membuat komponen UI dinamis
- **Blade** - Template engine bawaan Laravel
- **Tailwind CSS** - Framework CSS untuk styling
- **Alpine.js** - JavaScript ringan pelengkap Livewire

## Kontribusi
- Merancang dan membangun komponen Livewire untuk UI interaktif
- Membuat tampilan halaman utama menggunakan Blade template
- Mengimplementasikan real-time validation pada form input
- Membangun komponen reusable dengan Livewire
- Styling tampilan menggunakan Tailwind CSS
- Memastikan tampilan responsif di berbagai ukuran layar

## Komponen Livewire yang Dibuat
- `LoginForm.php` - Komponen form login dengan validasi real-time
- `Dashboard.php` - Komponen halaman dashboard utama
- `UserProfile.php` - Komponen halaman profil pengguna
- `DataTable.php` - Komponen tabel data dengan filter & pagination
- `SearchBar.php` - Komponen pencarian real-time
